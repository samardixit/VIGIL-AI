# VIGIL-AI
Vigil – AI-Based Smart Surveillance System
Vigil is focused on building an intelligent surveillance system capable of real-time monitoring and automated attendance tracking using facial recognition.

🔹 Features
🎯 Real-time face detection and recognition using OpenCV and DeepFace
📊 Automated student attendance system
⚡ FastAPI backend with asynchronous request handling
🔐 Secure authentication using JWT and bcrypt
🗄️ MySQL database integration with SQLAlchemy
🤖 Generative AI integration for intelligent insight

🛠️ Tech Stack
Python
FastAPI
OpenCV
DeepFace
MySQL
SQLAlchemy
JWT Authentication
NumPy
Google Generative AI

💡 Overview
This project demonstrates the integration of AI, backend systems, and databases to build a scalable and efficient real-world application. It highlights practical use of computer vision for attendance automation and system monitoring.
